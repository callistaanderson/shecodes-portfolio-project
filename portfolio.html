Weather App

This Weather App is my final project result from the SheCodes Plus workshop. Primarily Javascript, my Weather App accesses Open Weather Map API. This App has the ability to display current temperature, description, wind, and humidity of almost any city in the world. Additional features include: 3-hour temperature forecast, Fahrenheit/Celsius conversion, and current location weather display.

Vacation Rental

This is a multi-page project for a beautiful Vacation Home Rental. It utilizes the Bootstrap framework to make it responsive and eye-catching. This page displays the current (and often enviable!) temperature, using Open Weather Map API. It features many pictures and also includes a scrolling jumbotron on the homepage!
